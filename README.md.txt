# Valiant Spell Tracker

**Private Testing Repository** — For invited testers only.

Valiant Spell Tracker is an **unofficial** fan-made spell reference and character tool for *Tales of the Valiant*.

### ⚠️ Important Disclaimer
This is an **unofficial** third-party project.  
It is **not affiliated with, endorsed by, or connected to** Kobold Press or Open Design LLC.  
*Tales of the Valiant*, ToV, and all related trademarks are the property of Open Design LLC.

This repository is **private** and intended only for closed testing by a small group of trusted playtesters.  
All narrative flavor text has been removed or reworded. Mechanical content is based on open Black Flag rules where possible.

**Built by** a Game Master running a ToV campaign for my own table and the community.

### How to Test
- This repo is invitation-only
